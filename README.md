<h2 align="left">User-Manager CRUD App</h2>

Here's a simple project where I had pleasure to create my first concrete project using CRUD method and my own API with Node JS/Express and the Mongo database.

Front-end isn't really advanced, but back-end is functional. I tested it thanks to Postman.

In this project, you can create, update, find all, find one, update and delete users.
